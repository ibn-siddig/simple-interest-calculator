# Code of Conduct

## Our Pledge
We pledge to create a welcoming and respectful environment for everyone.

## Our Standards
Be respectful, inclusive, and professional.

## Enforcement Responsibilities
Project maintainers are responsible for enforcing the standards.

## Scope
This Code applies within all project spaces.

## Enforcement
Instances of abusive behavior may be reported.

## Enforcement Guidelines
Violations may result in warnings or bans.

## Attribution
This Code of Conduct is adapted from open source communities.
